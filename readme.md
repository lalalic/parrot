Requirement
===========
> As an english learner, I want to hear each chunk


Features
========
> Daily Topic
> Search Topic
> Print Topic
> Player
    > caption: on, off, number to delay some seconds
    > video: on, off
    > speed: 0.5 - 2
    > volume: on, off, number
    > pause: number to pause times of trunk duration
    > record: on, off, rate
    > select: flag current trunk
> Game
    > Shadowing
    > Dictating
    > Retell
